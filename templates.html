Tools​

  • backup-winstorage.sh – Backups storage folder in a docker container called windows.
  • backup-zfs.sh – Backups or restores zfs drive.
  • create-alias.sh – Create alias for command-line in proxmox.
  • create-lxc.sh – Creates an empty lxc container.
  • cron-reboot-ifdown.sh – Reboots proxmox if cloudflared lxc service is down.
  • docgen-templates.py – Auto-generate listing of scripts in specified folders.
  • ffmpeg-combinefiles.sh – Combines video files and set chapter markers inside docker container.
  • ffmpeg-convertmkv.sh – Converts mkv to mp4.
  • ffmpeg-convertwebm.sh – Convert webm to mp4.
  • fix-locale.sh – Fixes missing en_US.UTF-8 locale.
  • fix-pvenosub.sh – Configures no PVE subscription prompt and repositories.
  • format-disk.sh – Formats disk per user specification.
  • generate-lockfile.yml – Github Action manual workflow to generate one-off package-lock.json.
  • get-debian.sh – Downloads the latest debian ISO.
  • mediacms-import-addplaylist.sh – Worker script that adds a video playlist on mediacms.
  • mediacms-import-upload.py – Worker script executes the uploading of media files (mov, mp4, mkv) to mediacms.
  • mediacms-import.sh – Controller scripts that uploads all videos in a folder to a new playlist.
  • mediacms-initialise.sh – Installs the MediaCMS Docker container.
  • mount-drive.sh – Mounts a user-specified external drive and optionally updates fstab.
  • ocrmypdf-paperless-movefiles.sh – Transfer files in the output folder to paperless-ngx consume folder.
  • ocrmypdf-paperless-scanfiles.sh – Ocr scans files in the input folder to the output folder.
  • ocrmypdf-paperless.sh – Ocr scans files from ocrmypdf folder to paperless-ngx consume folder.
  • print-sysinfo.sh – Prints detailed system information.
  • print-workloads.sh – Prints all lxc, vms, dockers and services by their IPs, ports and status.
  • purge-dockerct.sh – Removes user-specified docker container(s) cleanly.
  • purge-lxc.sh – Removes a user-specified LXC container.
  • set-gpu.sh – Setups/switches nvidia gpu passthrough between vm and docker.
  • set-nonroot.sh – Switches from root to a non-root user and sets a default password.
  • set-swapfile.sh – Toggle swapfile to change available memory.
  • update-nvidiadriver.sh – Updates nvidia driver.
